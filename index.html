科研项目

命名对象网络

为探索原生支持网络内缓存和计算功能的无线网络协议，我们基于命名对象组网（NDN）开发针对网络内缓存、网络内处理的面向对象组网技术。通过改变因特网数据包的包头，并定义全新的转发表，实现网络内缓存-处理和通信资源的统一管理。 IEEE Internet of Things Journal. (JCR一区)

Github Repo

任意拓扑下联合缓存路由优化

在任意拓扑结构中的最优缓存是长期未得到解决的开放性问题。为了应对物联网等带宽受限的通信场景，我们开发了一种联合缓存路由技术，以平衡网络中不同路径上的流量，最大化了网络吞吐量，是目前第一个任意拓扑结构带宽约束下联合服务放置与路由优化策略。 IEEE Internet of Things Journal. (JCR一区)

Github Repo

任意拓扑下联合计算路由优化

众所周知，计算、缓存和通信资源的一体化调度属于NP-困难问题。我们找到了部分场景下物联网应用任务分配问题的最优解结构。针对一般场景，我们提出一种新型单时隙多维资源调度模型，并开发了联合优化算法JoCaPRAT，是目前效率最高的并行优化策略。 IEEE Internet of Things Journal. (JCR一区)
